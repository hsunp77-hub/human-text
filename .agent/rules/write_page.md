# Write Page (기록하기) 원칙

## 1. 텍스트 지침
- **타이틀**: Home과 동일하게 "오늘"을 유지합니다.
- **서브타이틀**: "당신 인생의 한순간을 떠올려보세요."

## 2. 기능 및 버튼 레이아웃
- **기록 전**: "기록" 버튼 하나만 노출됩니다 (비활성 상태).
- **기록 후**: "기록" 버튼과 "수정" 버튼이 **좌우**로 배치됩니다.
- **정렬**: 버튼 세트는 카드 영역 내에서 반드시 **가운데 정렬**되어야 합니다.
- **수정 기능**: "수정" 클릭 시 마지막 기록을 입력창으로 되돌리고 보관 내역에서 삭제합니다.

## 3. UI 정리
- "Reset" 문구 및 관련 초기화 기능은 제거된 상태를 유지합니다.
